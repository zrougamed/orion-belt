# SDKs

This directory is the home for language-specific Orion Belt client packages.

The HTTP contract comes from [docs/openapi/openapi.yaml](../docs/openapi/openapi.yaml), and the Go SDK in [pkg/sdk](../pkg/sdk) is the reference implementation for auth and request behavior.

## Layout

- [python](python) - Python client package
- [dotnet](dotnet) - .NET client package
- [js](js) - JavaScript / TypeScript client package

## Status

Go, JavaScript/TypeScript, Python, and .NET client scaffolds are now present.
